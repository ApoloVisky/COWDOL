# BACKEND

```sh
cd backend/
```

### Install the required libraries:
```sh
pip install -r requirements.txt
```

### Run the script:
```sh
fastapi run server.py --host 127.0.0.1

```

# FRONTEND
```sh
cd frontend/
```
### Install the required libraries:
```sh
npm install
```

### Run the script:
```sh
npm run start
```
